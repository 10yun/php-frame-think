# php-frame-think


## 介绍

基于thinkphp6扩展  
支持 thinkphp6（tp6）php8原生（Attribute）注解   
Support thinkphp6 (tp6) php8（Attribute）annotation 

## 依赖

- php8 | php7
- thinkphp6


## 目录结构

~~~
|- addons
    |- moduleA             # -- 模块A
        |-  config          # 配置
        |-  controller      # 控制器
        |-  models          # 模型
        |-  route           # 路由
        |-  services        # 服务

    |- moduleB             # -- 模块B
        |-  config          # 配置
        |-  controller      # 控制器
        |-  models          # 模型
        |-  route           # 路由
        |-  services        # 服务
|- app
|- config
|- public
|- route
|- vendor
~~~
